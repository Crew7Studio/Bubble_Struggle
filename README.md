# BUBBLE TROUBLE

This is a replica of the game Bubble Trouble 

## My project at itch.io link - https://crew7studiogames.itch.io/bubble-trouble
## My itch.io page link - https://crew7studiogames.itch.io/
## Gameplay link - https://youtu.be/SiNyNscHsMs

### FEATURES : 
* Random PowerUp spawner spawns power ups for  New Life and Rapid Fire
* Can Jump on to platforms and shoot

### CONTORLS : 
* Use A/D or Left/Right arrows to Move
* Use Space to Jump
* Use Left Ctrl or Left mouse button to Fire
* Press P or ESC to go to Pause menu


### TOOLS USED
* Unity 2019.1
* Adobe photoshop CC
